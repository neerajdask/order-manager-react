# Welcome to the Order Manager App

This project was developed using the React.js library. The UX is obtained and composed from Material UI components [https://material-ui.com/getting-started/installation/]. 

## Packages Used

    1. Material UI
    2. Firebase
    3. History
    4. Axios
    5. Redux
    6. Redux-Thunk
    7. Redux- Form
    8. React Router DOM
